# Welcome to MD Reader

This is a **test document** to verify the MD Reader application works correctly.

## Features

1. **File Loading** — Open any `.md` file and see it rendered
2. **Text-to-Speech** — Listen to your documents read aloud
3. **Speed Control** — Adjust playback from 0.5x to 2x
4. **Voice Selection** — Choose from macOS system voices
5. **AI Transform** — Rewrite content in different styles

## A Short Story

Once upon a time, in a land of markdown files, there lived a developer who was tired of reading documentation. "If only," they thought, "someone could read this to me while I make coffee."

And so, MD Reader was born.

## Technical Details

Here's some code that demonstrates the concept:

```javascript
const reader = new MDReader();
reader.open('document.md');
reader.play({ speed: 1.5, voice: 'Samantha' });
```
